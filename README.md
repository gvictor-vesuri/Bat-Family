# 🦇 Bat Family

Projeto web desenvolvido com HTML, CSS e JavaScript, com foco em interface visual e organização de conteúdo.

## 🚀 Funcionalidades
- Exibição de personagens da Bat Family
- Layout organizado e visualmente atrativo
- Navegação simples entre seções
- Interface responsiva

## 🛠️ Tecnologias
- HTML
- CSS
- JavaScript

## 🎯 Objetivo do projeto
Este projeto foi desenvolvido com o objetivo de praticar desenvolvimento front-end, incluindo estruturação de páginas, estilização com CSS e organização de conteúdo.

## 📚 Aprendizados
- Estruturação de páginas com HTML
- Estilização e layout com CSS
- Organização de conteúdo em interface web
- Noções básicas de interatividade com JavaScript
